Hello MarkForth

This document was written in MarkForth. With MarkForth you write a minimal input file using familiar HTML tags to define the semantic structure of the document only. MarkForth will convert the input into a single HTML file that leverages the full power of HTML, CSS and JS.

It's like HTML and Markdown had a baby!

Features

Automatic Headers

Organise your content into <section> elements. Define all your headers in <h1> tags. MarkForth will automatically determine which header style to use based on nesting. You will never need to go through your document changing <h2> to <h3> because you grouped some of your sections together.

Component System

MarkForth has a fully customisable component system. Some pre-authored components are described below.

Table of Contents

This component numbers all the section and subsections and generate a table of contents as seen at the top of this document.

Code Blocks

This component displays blocks of code. There is a button to copy the code to the clipboard in the top-right corner.

Image Embedding

When HTML documents reference external images on the local filing system, those images need to be transported with the document to be available when viewed from elsewhere. MarkForth automatically embeds images in the output HTML for ultimate portability.
